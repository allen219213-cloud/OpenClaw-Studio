# 用户手册

## 1. 登录与用户管理
- 默认管理员：`admin / admin123`
- 在“协作分享”页可创建普通用户和管理员。

## 2. 工具市场
- 查看官方工具和第三方工具。
- 点击“评论”查看评分与反馈。
- 填写工具信息后可一键安装第三方工具。

## 3. 任务编排
- 进入“任务编排”页，使用拖拽节点构建工作流。
- 支持节点类型：开始、结束、智能体、条件、循环、并行。
- 使用节点属性面板配置参数。
- 可导入导出 JSON/YAML。

## 4. 任务执行
- 在“任务执行”页启动、暂停、继续、停止、重试。
- 查看实时日志和智能体对话。
- 结果可导出 Markdown/PDF/HTML。

## 5. 协作与社区
- 可创建公开/私有分享链接。
- 可上传社区模板并下载他人模板。
