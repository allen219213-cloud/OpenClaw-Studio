# 快速入门视频脚本

## 片头（10s）
- 展示 OpenClaw-Studio 首页，说明目标：5 分钟完成从编排到执行。

## 第一步：创建任务（60s）
- 打开“任务编排”页面。
- 拖拽开始/智能体/结束节点并连线。
- 配置智能体节点参数。
- 保存任务。

## 第二步：执行任务（60s）
- 打开“任务执行”页面。
- 启动任务，展示进度条和实时日志。
- 展示暂停/继续和重试。

## 第三步：导出结果（30s）
- 导出 Markdown/PDF/HTML。
- 展示导出路径。

## 第四步：协作分享（40s）
- 创建公开分享链接。
- 上传社区模板并下载。

## 结尾（10s）
- 引导用户查看用户手册与开发者文档。
