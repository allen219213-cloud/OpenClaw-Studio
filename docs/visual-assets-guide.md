## 三、存储规范

- 命名规范：
  - `screenshot-1-main.png`
  - `screenshot-2-core.gif`（如果是动图）
  - `compare-old-vs-new.png`
- 存储路径：`assets/screenshots/`
- 体积优化：
  - PNG 建议 < 400KB
  - GIF 建议 < 8MB
  - 优先保证 GitHub 打开速度和移动端加载体验
- 建议工具：
  - 图片压缩：TinyPNG / Squoosh
  - GIF 录制：ScreenToGif / Kap
